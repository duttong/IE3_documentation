## Routine Operating Pattern

1. Start the software with the desktop `IE3 Run` launcher.
2. Monitor chromatograms and daily status values.
3. Record cylinder pressures weekly.
4. Use the operator log for changes and observations.
5. Let the daily sync job send recent data and logs.

For routine work, prefer the controls in the running application over manually editing files.
